About

Hey there! This page is included as an example. If you’re interested in learning more about how to use Hugo, take a look at the docs. If you want to learn more about the differences between Jekyll and Hugo, take a look here.

In the novel, The Strange Case of Dr. Jeykll and Mr. Hyde, Mr. Poole is Dr. Jekyll’s virtuous and loyal butler. Similarly, Poole is an upstanding and effective butler that helps you build Jekyll themes. It’s made by @mdo. The whole thing was then ported over to Hugo by @ciarand.

There are currently two themes built on Poole:

  • Hyde

  • Lanyon

Learn more and contribute on GitHub.

Setup

Some fun facts about the setup of this project include:

  • Built for Jekyll, ported to Hugo

  • Developed on GitHub and hosted for free on GitHub Pages

  • Originally coded with Sublime Text 2, an amazing code editor

  • Ported in vim, a thoroughly obtuse code editor for masochists

  • Originally designed and developed while listening to music like Blood Bros Trilogy

  • Ported while listening to this one song on repeat.

Have questions or suggestions? Feel free to open an issue on GitHub or ask @mdo on Twitter.
